# 📜 mnzproject

mnzproject is a web-based platform that allows users to scroll through scientific research papers, similar to TikTok. The recommendation algorithm adapts based on the papers users like and save, providing a personalized experience.

## ✨ Key Features
- 🔍 **Paper Recommendations** - Displayed papers align with user interests.
- 🎯 **User Interactions** - Users can like, save, and share research papers.
- 📜 **Infinite Scrolling** - Seamless navigation for endless exploration.
- 🏷 **Categories & Tags** - Users can search for papers based on categories or tags.
- 📊 **User Analytics** - View trending papers and personalized suggestions.

## 🛠 Technologies Used
- **Frontend:** React
- **Backend:** Node.js
- **Database:** Supabase

## 🚀 How to Run the Project

1. Clone this repository:
   ```sh
   git clone https://github.com/username/mnzproject.git
   ```
2. Navigate to the project directory:
   ```sh
   cd mnzproject
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
